About Flavor Archive

Flavor Archive is an open-source recipe manager built to help you organize, discover, and share your favorite dishes. Whether you're cataloging family heirlooms or experimenting with new cuisines, it keeps everything in one place.

This project is developed in the open and released as free software. Contributions of all kinds — code, recipes, ideas — are welcome.

  • Share it — with friends, family, and anyone who loves to cook
  • Contribute code — submit pull requests on GitHub
  • Add recipes — help grow the collection
